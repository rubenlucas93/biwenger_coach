# Biwenger Optimizer App

A mobile application designed to optimize your Biwenger fantasy football team.

## Overview

This project is built with **React Native** and **Expo**. It includes logic for scraping and interacting with the Biwenger API to provide team optimization recommendations.

### Key Features
- **Biwenger Integration:** Connects to your Biwenger account.
- **Team Difficulty Analysis:** Evaluates match difficulty for your players.
- **Optimizer:** Suggests the best possible lineup based on various metrics.

## Development

To start the development server:

```bash
npm install
npm run start
```

## Building the App

To build the mobile application for Android using EAS Build, run:

```bash
eas build --platform android --profile preview
```
